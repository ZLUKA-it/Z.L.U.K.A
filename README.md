# Z.L.U.K.A

## 🚀 Возможности

### 📸 Управление и мониторинг
- **Скриншоты** — мгновенный снимок экрана сервера (устройства, с которого запущен бот) с отправкой в Telegram-чат
- **Буфер обмена** — просмотр содержимого буфера обмена сервера удалённо
- **Мониторинг дисков** — информация о занятом месте на всех дисках

### 🛡️ Стабильность и безопасность
- **Автоперезапуск** — при критических ошибках бот перезапускается автоматически
- **Логирование** — все ошибки и вывод программы сохраняются в папку `logs` с указанием времени
- **Управление логами** — автоматическое ограничение размера логов в МБ и удаление старых файлов
- **Анти-спам** — защита от флуда (ограничение количества сообщений в минуту/час)

### 🔧 Гибкость и адаптивность
- **Режимы работы** — запуск как в открытой консоли (для отладки), так и в фоновом режиме (скрыто)
- **Автозагрузка** — возможность запуска вместе с системой (через планировщик или автозапуск)
- **Обход блокировок** — поддержка прокси (по умолчанию порт 12334 для Hidify / HTTP/SOCKS5)

### 📋 Управление
- **Команды и меню** — интуитивное меню с кнопками для быстрого доступа к основным функциям
- **Медиа-контроль** — управление плеером (громкость, воспроизведение, треки)
- **Управление ПК** — перезагрузка, выключение, гибернация, блокировка экрана
